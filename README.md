🌿 Leaf Disease Detection System

This project is an image processing–based machine learning system designed to automatically detect and classify plant leaf diseases. The model is trained on the well-known PlantVillage Dataset, which contains over 40,000 images of healthy and diseased plant leaves across multiple species.

🚀 Overview

Early detection of plant diseases is crucial for improving crop yield and reducing agricultural losses. This project leverages computer vision techniques and deep learning to identify diseases directly from leaf images.

🧠 Model Details
Trained on 40,000+ labeled images
Multi-class classification (different crops and diseases)
Uses image preprocessing and feature extraction techniques
Built using deep learning (CNN-based architecture)
⚙️ Features
Detects multiple plant diseases from leaf images
Classifies both healthy and infected leaves
Fast and automated prediction system
Can be extended to real-time applications (mobile/web)
🛠️ Tech Stack
Python
OpenCV (image processing)
TensorFlow / PyTorch (model training)
NumPy, Matplotlib
📊 Dataset
Source: PlantVillage Dataset
Contains images of various crops like tomato, potato, corn, etc.
Includes both healthy and diseased leaf samples
